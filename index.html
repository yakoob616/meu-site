Sobre o Jogo

Inspirado no xadrez, o Batalha dos Cogumelos é um jogo de tabuleiro estratégico que combina peças e cartas de poderes especiais. Cada jogador comanda um exército de cogumelos mágicos, batalhando para capturar o rei adversário.

Como Jogar

O jogo é para 2 jogadores, cada um controlando um conjunto de peças e um deck de cartas.

  • Peças: Movimentam-se como no xadrez, mas possuem habilidades especiais.
  • Cartas de Poder: Use habilidades mágicas para ganhar vantagem sobre o oponente.
  • Objetivo: Capture o "Rei Cogumelo" do oponente para vencer.

Cartas de Poder

Use cartas de poder para modificar as regras do tabuleiro, atacar diretamente ou defender suas peças. Cada carta possui um efeito único!

Explosão de Esporos

Atordoa as peças próximas por um turno.

Escudo Mágico

Protege uma peça de qualquer ataque por um turno.

Movimento Ágil

Permite mover uma peça duas vezes.
